# Компонент Weather — Прогноз погоди на 16 днів (React)

Цей React-компонент відображає 16-денний прогноз погоди для Одеси, отримуючи дані з бекенду через API-запит.

## Опис роботи

- Використовує хуки `useState` та `useEffect` для управління станом та завантаженням даних.
- При монтуванні компонента виконується запит на `/api/realweather/odessa` (шлях проксі налаштований у `vite.config.js`).
- Обробляє стани: завантаження, помилка та успішне отримання даних.
- Відображає прогноз у сітці карток з інформацією про дату, іконку, опис, температуру, опади та вітер.
- У футері вказується джерело даних (Open-Meteo) та час оновлення за київським часовим поясом.
