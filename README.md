# Компонент Weather — Прогноз погоди в Одесі на 16 днів (React + ASP.NET Core)

Цей React-компонент відображає 16-денний прогноз погоди для Одеси. Дані отримуються з бекенду ASP.NET Core через API-запит.

## Опис роботи компонента

- Використовує хуки `useState` та `useEffect` для управління станом та завантаженням даних.
- При монтуванні компонента виконується запит на відносний шлях **`/api/realweather/odesa`**.
- У режимі розробки Vite (React-клієнт) автоматично проксірує цей запит на ASP.NET Core бекенд завдяки налаштуванням у `vite.config.js`.
- У production (після `npm run build`) статичні файли React розміщуються в ASP.NET Core проєкті, тому запит іде безпосередньо на той самий сервер — без CORS-проблем.
- Компонент обробляє три стани:
  - Завантаження (`loading`)
  - Помилка (`error`)
  - Успішне отримання даних
- Прогноз відображається у сітці карток з інформацією про дату, іконку погоди, опис, максимальну/мінімальну температуру, ймовірність опадів та швидкість вітру.
